# Business Consultancy App

A modern, full-stack business consultancy web application
built with Next.js 15 and Tailwind CSS. Designed to present
consultancy services, case studies and client engagement
in a professional, responsive format.

## Features
- Modern responsive UI built with Next.js App Router
- Tailwind CSS styling with dark mode support
- Fast page loads with Next.js server-side rendering
- Optimized for desktop and mobile
- Clean component architecture with TypeScript

## Tech Stack
Next.js 15, React, TypeScript,
Tailwind CSS, ESLint

## Run Locally
npm install
npm run dev

Open http://localhost:3000

## Build for Production
npm run build
npm start
